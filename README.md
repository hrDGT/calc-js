# How to run the app

* Install dependencies using `npm i`

* Create a build with the command `npm run build`

* Run the application using `npm run start`

# Project structure

* The project root contains its folders and configuration files

* The .husky folder contains the pre-commit hook configuration

* The src folder contains the project development files.

* After installing the dependencies, a node_modules folder appears containing the modules to be installed

* After building the project, a dist folder appears containing the built version of the project
